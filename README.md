# Kinvey PHP

[![Build Status](https://travis-ci.org/italoqueiroz/kinvey-php.svg)](https://travis-ci.org/italoqueiroz/kinvey-php)

## Install

Via Composer

``` bash
$ composer require izie/kinvey-php
```

## Usage

``` php
\League\Kinvey\Kinvey::initialize('<app_id>', '<app_secret>', '<master_secret>');
```

## Testing

``` bash
$ phpunit
```
